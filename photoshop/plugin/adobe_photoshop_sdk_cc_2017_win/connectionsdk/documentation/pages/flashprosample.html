Flash Professional CS5 Example

Requirements

  • Flash Professional CS5
  • AIR for Android Extension
  • 's AS3 swc (PhotoshopSDK.swc included in folder)
  • StructuredLogSDK (StructuredLogSDK.swc included in folder)

Overview

This Flash CS application demonstrates how to use the AS3 sdk to build an application that controls the active Photoshop tool. It shows how to make a connection to Photoshop and send encrypted commands to the tool.

How to use this project

Open the .fla file in Flash Professional CS5 or higher and publish the application to a device. Alternatively, you can try running it on the desktop.
