Required libraries

  • as3corelib (https://github.com/mikechambers/as3corelib)
  • as3crypto (https://code.google.com/p/as3crypto/)
  • StructuredLogTestingSDK (https://code.google.com/p/structuredlogtesting/)

Overview

If you're compiling the yourself, you can either include the source paths from the above libraries, or drop the SWC files into the "libs" folder of your project.

Code is in src/com/adobe/photoshop/
SWC is bin/PhotoshopSDK.swc
