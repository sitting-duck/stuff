Creating Android apps with Adobe AIR

Requirements

  • Flash Builder 4.5
  • Flex 4.5
  • AIR 2.6
  • 's AS3 swc (PhotoshopSDK.swc) located in samples/air/as3sdk

Overview

This is a basic Photoshop layer comp viewing application. Typically, a designer mocks up an application as a Photoshop document. The designer uses layer comps to show multiple ideas for the application. The device comp viewer application will show these multiple ideas on the target device.

How to use this project

Project name: DeviceCompViewer.fxp

In Flash Builder 4.5, click on Import > Flash Builder Project. Select "DeviceCompViewer.fxp" and click OK. The project is set up and you can now either run it on the desktop or package it up for a mobile target.

The sample code is located in the the com.adobe.example.* package.
